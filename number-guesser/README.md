# NumberGuesser - front

This is an interface of this [backend](https://github.com/thaynnara007/numberGuesser)

---

## Usage 

> Let's enter in the directory
```shell
$ cd number-guesser
```
> Let's install the dependencies
```shell
$ npm install
```
> Let's start the app
```shell
$ npm start
```
```shell
$ Runing on http://localhost:3000
```
